## Summary
This task involved analyzing phishing email samples, identifying red flags, and creating a professional awareness report for employees.

## Analysis Summary

| Sample | Classification | Key Red Flags |
|--------|----------------|---------------|
| Account Locked | 🔴 PHISHING | Fake PayPal domain, urgency, malicious link |
| CEO Request | 🟡 SUSPICIOUS | Gift card scam, spoofed CEO email |
| Invoice Attachment | 🔴 PHISHING | .pdf.exe attachment, fake vendor domain |

## Key Learnings
- Identifying phishing indicators (spoofed domains, urgency tactics)
- Using email header analyzers to detect spoofing
- Creating security awareness content for non-technical users
- Risk classification of suspicious emails

## Prevention Guidelines
- ✅ Hover over links before clicking
- ✅ Verify sender email addresses
- ✅ Report suspicious emails to IT
- ❌ Never open unexpected attachments
- ❌ Never share passwords or OTPs
